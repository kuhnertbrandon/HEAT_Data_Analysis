HEAT Data Analysis Script

Procedure to Run
1. Pull data from Instronita or Benderita
2. ENSURE title matches this format XXXXXXX_SNyyy. If it does not match this, most if not all the code will break and not file correctly
    This will get updated at some point
3. Drop the HEAT_Data_Analysis.py and run using python 
4. Sit back and enjoy as the plots are made for you

  Resistance vs. Cycle plots
